logMyRun()

Main function of the app. Fills in with zeros any optional fields left blank, calls pad2 to add leading zeros to some single-digit fields, calculates the pace of the run and 'logs' it by adding data to the DOM. Calls formReset to clear the form and toggles the observable showTable to 'true' to display the previously hidden data div.
